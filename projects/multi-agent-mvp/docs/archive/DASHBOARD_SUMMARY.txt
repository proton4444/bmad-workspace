    "description": "Test",

    "agent_names": ["Athena", "Cato"],
    "problem_statement": "Design a feature"
  }'

Terminal 3: Execute Workflow

Browser: Open Dashboard
Visit http://localhost:3000

WEBSOCKET MESSAGE PROTOCOL
==========================

Format:
{
  "type": "task_update|agent_update|idea_added|phase_changed",
  "workflow_id": "workflow-id",
  "data": { /* event data */ },
  "timestamp": "ISO-8601"
}

Event Types:
1. task_update: Task status changed
2. agent_update: Agent state changed
3. idea_added: New collaborative idea
4. phase_changed: Workflow phase transitioned

SUPPORT & TROUBLESHOOTING
========================

Dashboard shows "No workflows found"
  → Create a workflow using curl command
  → Check API server is running

Connection shows "Offline"
  → Verify port 8000 is accessible
  → Check FastAPI server is running
  → Refresh the page

Tasks not showing in DAG
  → Wait a moment for data to load
  → Check browser console (F12) for errors

Ideas timeline is empty
  → Execute workflow to generate ideas
  → Refresh page to reload ideas

See DASHBOARD_README.md for full troubleshooting guide.
